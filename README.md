## Installation

To get started with the TikTok Mass Reporting Bot, follow these steps:

1. **Download the Bot**: Head over to the [Releases](https://github.com/Manishh8595/tiktok-mass-reporting-bot/releases) section. Download the latest version of the bot.
   
2. **Extract the Files**: Once downloaded, extract the files to your desired location.

3. **Install Dependencies**: Open your terminal and navigate to the extracted folder. Run the following command to install the required dependencies:

   ```bash
   npm install
   ```

4. **Configuration**: Before running the bot, you may need to configure certain settings. Open the `config.json` file and adjust the parameters as needed.

5. **Run the Bot**: To execute the bot, use the following command:

   ```bash
   node index.js
   ```

## Usage

Once the bot is up and running, you can start using it to report content. Here's how:

1. **Input TikTok Video Links**: Prepare a list of TikTok video links that you want to report. You can input these links directly into the bot interface.

2. **Set Reporting Criteria**: Define the reasons for reporting. The bot offers various options, so choose the most relevant ones.

3. **Start Reporting**: Click the "Start Reporting" button. The bot will begin processing the links and report them according to your specified criteria.

4. **Monitor Progress**: The bot provides real-time updates on the reporting process. You can monitor the status and see how many reports have been successfully submitted.

5. **Review Reports**: After the process is complete, you can review the report logs to see which videos were reported and the reasons behind each report.
